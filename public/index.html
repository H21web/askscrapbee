This is a serverless API for scraping content from iask.ai using Puppeteer.

Usage:

GET /api/scrape?query=your_question

Examples:

Example 1: /api/scrape?query=What is artificial intelligence?
Example 2: /api/scrape?query=How does machine learning work?

Response Format:

{
  "success": true,
  "text": "Extracted content from iask.ai...",
  "query": "your_question",
  "url": "https://iask.ai/q?mode=question&options[detail_level]=concise&q=...",
  "timestamp": "2025-08-20T08:25:00.000Z",
  "source": "puppeteer",
  "textLength": 245
}

Error Response:

{
  "error": "Error type",
  "details": "Error description",
  "query": "your_question"
}
